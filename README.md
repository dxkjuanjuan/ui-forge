# 🎨 ui-forge — Skills & Showcase

## Showcase

A live demo page with 11 brand themes (Apple, Claude, Cursor, Nvidia, Tesla, Stripe, Vercel, Linear, Spotify, Notion + default), GSAP animations, particle effects, and real-time brand switching.

Open `showcase/index.html` in a browser.
